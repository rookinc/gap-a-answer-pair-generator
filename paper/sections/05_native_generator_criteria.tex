\section{Native generator criteria}

A successful native generator must satisfy the following conditions.

\begin{enumerate}[leftmargin=2em]
\item It must construct a candidate universe without using realized pair labels as input.
\item It must be expressed in source-native fields or in a justified native quotient structure.
\item It must produce a candidate universe broad enough for false positives to be possible.
\item The reciprocal answer-pair selector must recover exactly 12 pairs.
\item The final selector profile must have zero false positives and zero misses.
\end{enumerate}

A search that only re-encodes the 12 realized pairs is not sufficient. A search that depends on the admitted label is not sufficient. A search that uses WXYZTI row identity as the generator is not sufficient.

The desired result is an admission-blind native generator followed by an exact reciprocal selector.
